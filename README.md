Atma Laundry is a web application that provides information about Atma Laundry services, including details about services, location, reviews, and more. This platform allows both clients and admins to log in, manage and create laundry orders, as well as handle transactions such as service types, categories, discounts, and other related features.

---

## Features
- **Client and Admin Login**: Secure login for both clients and admins to manage and place orders.
- **Order Management**: Admins can manage orders, while clients can create and track their laundry orders.
- **Service Options**: Clients can choose from various laundry services, categories, and discounts.
- **Location & Reviews**: Information about Atma Laundry's location and customer reviews.
  
## Tech Stack
- **Backend**: Laravel (PHP Framework)
- **Frontend**: HTML, CSS, JavaScript
- **Database**: MySQL (Database can be imported from the provided `database_laundry.sql` file)
